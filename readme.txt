Add UI to coexist with Gutenberg happily.

== Description ==

This plugin adds these UI below only when Gutenberg is active.

1. Adding 'Classic Editor' option under 'Add New' button in post.php screen including custom post type.
2. Adding 'Add new (classic)' link to admin menu including custom post type.
     - You can change the menu text by using hook 'cwg_post_new_classic_menu_text'.
3. Adding 'Switch to (Classic|Gunteberg) editor' button to admin bar including custom post type.

== Installation ==

1. Upload the 'coexist-with-gutenberg' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the Plugins menu in WordPress.
